# CS-StudyAbroad(平民版)
相信点进来的朋友，应该都有过类似的想法：工作几年，对工作厌烦了，生活像一潭死水。  

想通过留学的方式见见更大的世界，但是现状不太好。可能是:
- 本科学校一般
- 绩点一般
- 手头积蓄一般
- 英语水平一遍
- ...
  
然后这个仓库希望共建一个平民留子可选的院校集合，供大家参考。收录的学校要求底线是非野鸡大学，其次满足任一即可：
- 申请难度不苛刻
- 留学成本低
